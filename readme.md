# Requirements
| Name                    | Version |
| ----------------------- | :-----: |
| Python                  | Latest  |
| Browser(Chromium Based) | Latest  |

# Installation

1. install latest python
2. install all requirements package
    ```bash
    pip install requirements.txt
    ```
3. set FLask App ENV
   - Windows CMD
    ```bash
    set FLASK_APP=app
    ```
    - Terminal BASH
    ```bash
    export FLASK_APP=app
    ```
4. run apps
    ```bash
    flask run
    ```
